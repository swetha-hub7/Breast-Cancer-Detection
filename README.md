# Breast Cancer Detection Using Machine Learning

## Overview
A Machine Learning web application that predicts whether a breast tumor is Benign or Malignant using patient features.

## Technologies Used
- Python
- Flask
- Scikit-Learn
- HTML
- CSS
- JavaScript

## Features
- Real-time prediction
- Confidence score
- Responsive dashboard
- Machine Learning model
